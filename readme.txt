A simple Divi module to display a logo slider on your website with centralized logo management from WordPress admin.

== Description ==

This plugin adds a new module to the Divi Builder called "Logo Slider". You can use this module to display a slider of logos from your partners, clients, or sponsors. The module is easy to use and comes with a variety of settings to customize the slider to your needs.

**Key Features:**

* **Centralized Logo Management** - Upload and manage all your logos from the WordPress admin dashboard
* **Two Logo Sources** - Choose between admin-managed logos or add custom logos directly in the Divi Builder
* **Drag & Drop Reordering** - Easily reorder your logos with drag and drop functionality
* **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
* **Customizable Settings** - Control slides per view, spacing, speed, autoplay, and navigation
* **SEO Friendly** - Add alt text and links to your logos for better accessibility and SEO

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to "Logo Slider" in your WordPress admin menu to upload and manage your logos.
4. Create a new page or edit an existing one with the Divi Builder.
5. Add the "Logo Slider" module to your page.
6. Choose to use admin-managed logos or add custom logos directly in the builder.
7. Configure your slider settings and publish!

== Frequently Asked Questions ==

= How do I add logos to the slider? =

You have two options:
1. **Admin-Managed Logos**: Go to "Logo Slider" in your WordPress admin menu to upload and manage logos centrally. Then select "Use Admin Managed Logos" in the Divi module and choose which logos to display.
2. **Custom Logos**: Select "Add Custom Logos" in the Divi module and add logos directly within the builder.

= Can I reorder my logos? =

Yes! In the admin dashboard, you can drag and drop logos to reorder them. The order will be reflected in your sliders.

= Can I use the same logos across multiple pages? =

Absolutely! When you use admin-managed logos, you can reuse the same set of logos across multiple pages and sliders.

= Can I change the number of logos displayed at once? =

Yes, you can change the number of logos displayed at once by editing the "Logos per View" setting in the module settings.

= Can I edit or delete logos after adding them? =

Yes, in the admin dashboard you can edit logo details, reorder them, or delete them as needed.
